Use this form to add a new task to your daily to-do list

Give your task a short and snappy name, some (optional) notes, and assign it a difficulty.

How does difficulty work?

Level Up features 4 ranks of difficulty; Easy, Medium, Hard, and Ambitious. The harder a task, the more experience you gain when you complete it, but the more hitpoints you'll lose if you fail to complete it at the end of the day.

REMEMBER: Make sure the task you add is a task you want to do every day. The ability to add tasks that can be done on set days, or one-off tasks, will be included in a future version of Level Up.
